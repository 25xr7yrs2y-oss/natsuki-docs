# EU Customs Legal Knowledge Base

This wiki is a structured legal and operational source base for EU customs and cross-border trade compliance. It is written for human reading, machine translation, and future knowledge-base ingestion.

The source check date for this wiki is 2026-06-11. Dynamic sources such as TARIC, quota balances, sanctions, Safety Gate alerts, TRACES controls, and product-specific document codes must be queried again for each transaction.

## How to Use This Wiki

Start with the EU-level legal framework when the question is about legal authority. Use the procedure pages when the question is about moving goods. Use the dynamic-data page before relying on any duty rate, quota, anti-dumping measure, certificate, licence, or product restriction.

This wiki separates EU-level law, official operational guidance, dynamic database results, and Member State implementation.

## Core Sources

The core legal sources are EUR-Lex, ELI links, CELLAR SPARQL metadata, DG TAXUD guidance, TARIC, Access2Markets, EU Customs Trader Portal systems, Member State customs authorities, WTO, WCO, and official EU product-control portals.
